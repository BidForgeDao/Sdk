**Win the auction. Define the token. Earn the rewards.**

BidForge introduces a new way to launch tokens: instead of predefined rules, the community competes to define them. The highest bidder earns the right to set the token’s details and receives a share of the trading fees.

---

## ✨ Overview

BidForge is a decentralized auction-based token creation platform where:

- 🏆 The auction winner defines the token
- ⚙️ Token details are set after the auction ends
- 💰 Winners earn a share of trading fees
- 🔍 Everything is transparent and on-chain

---

## ⚡ How It Works

1. **Auction Starts**
   - A new token slot opens for bidding

2. **Users Place Bids**
   - Participants compete to win control

3. **Winner Decides**
   - Token name, ticker, and parameters are set by the winner

4. **Token Launch**
   - The token is deployed and becomes tradable

5. **Earn Rewards**
   - The winner receives a portion of trading fees
